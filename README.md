# 🚀 SmartCommute

### AI-Based Multi-Modal Route Optimization & Carpool Matching System

SmartCommute is a modern web application designed to optimize daily commuting in metro cities by combining multiple transport modes and providing intelligent route suggestions. It also simulates a smart carpool matching system to reduce travel time, cost, and congestion.

---

## 🌟 Features

### 🗺️ Multi-Modal Route Optimization

* Suggests multiple routes:

  * ⚡ Fastest
  * 💰 Cheapest
  * 🧘 Least Crowded
* Displays time, cost, distance, and transport modes

---

### 🤖 Smart Recommendation Engine

* Rule-based AI logic to recommend routes based on:

  * Time
  * Cost
  * Comfort
* Highlights the best route dynamically

---

### 🤝 Carpool Matching System

* Displays available carpools with:

  * Name
  * Route
  * Timing
  * Seats available
* “Request Ride” UI simulation

---

### 📊 Commute Analytics Dashboard

* Weekly time saved
* Cost savings
* CO₂ emissions reduced
  *(Simulated data for demonstration)*

---

### 🌙 Modern UI/UX

* Clean and responsive design
* Dark mode toggle
* Smooth animations and transitions
* Interactive cards and dashboards

---

### 🧠 Advanced Features

* Recent searches (localStorage)
* Favorite routes ⭐
* Route filtering (time/cost)
* Loading skeletons & notifications

---

## 🛠️ Tech Stack

* **Frontend:** React (Hooks, Functional Components)
* **Maps:** Leaflet.js (OpenStreetMap)
* **Data:** Static JSON (No backend required)
* **Deployment:** Vercel

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/smartcommute.git
```

2. Navigate to project folder

```bash
cd smartcommute
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm start
```
